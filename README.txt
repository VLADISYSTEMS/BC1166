# BC1166 Página Web

## Instrucciones para publicar en GitHub Pages

1. Ve a [GitHub](https://github.com/) e inicia sesión.
2. Crea un repositorio nuevo llamado `BC1166`.
3. Sube todos los archivos de esta carpeta al repositorio (incluyendo la carpeta img).
4. Ve a **Settings** → **Pages**.
5. En *Source*, selecciona `main` y `/ (root)`.
6. Guarda y espera 1-2 minutos.
7. Tu página estará online en: `https://TUUSUARIO.github.io/BC1166`.

## Cómo cambiar el contenido
- **Texto**: Edita `index.html` desde GitHub o tu PC.
- **Imágenes**: Cambia los archivos en la carpeta `img` manteniendo el mismo nombre.
- **Colores**: Busca en `index.html` los códigos hexadecimales (ej: `#7E3DAF`) y cámbialos.
